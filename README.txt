== AMEEMaterials

A prototype Autodesk Inventor addin that uses AMEEconnect to show 
environmental data for materials during design.

Licensed under the BSD 3-Clause license (See LICENSE.txt for details)

Authors: James Smith, Adam Nagy

Copyright: Copyright (c) 2011 AMEE UK Ltd

Homepage: http://github.com/AMEE/inventor

== INSTALLATION

No installation package is currently supplied, but you can build the code
using Visual C#.

== REQUIREMENTS

 * Autodesk Inventor 2012
 * RestSharp (http://restsharp.org/)
   - DLL is currently included in the source for convenience.

See the video at http://www.youtube.com/watch?v=kvBtx8KMSWE for usage details.
